# 용어 사전 (Context)

이 문서는 LongformShorts AI 도메인의 공통 언어를 정의한다. 구현 세부사항이
아니라 "무엇을 무엇이라 부르는가"만 담는다.

## 프로젝트 (Project)

사용자가 업로드한 하나의 원본 롱폼 영상과, 그로부터 파생된 전사·캡션·쇼츠
전부를 묶는 단위. 업로드 시점에 콘텐츠 타입과 언어가 정해지며, 전사 → 쇼츠
구간 탐지 → 클립 렌더로 이어지는 파이프라인의 상태(`status`)를 가진다.

## 쇼츠 (Short)

하나의 프로젝트에서 잘라낸 세로형 짧은 클립 하나. 시작/끝 구간, 자체 캡션,
캡션 스타일, 렌더된 클립을 가진다. 한 프로젝트는 여러 쇼츠를 가진다.

## 생성 기본값 (Generation Defaults)

새 프로젝트나 쇼츠를 만들 때 **시작점이 되는 사용자별 값**. 강제 적용이
아니라 **사전 채움(pre-fill)** 이다 — 설정에서 정해두면 입력 화면의 초기값을
채울 뿐, 사용자는 그 자리에서 다르게 고를 수 있고 최종 진실은 각 프로젝트/
쇼츠에 저장된 값이다.
